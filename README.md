高光谱图像处理库 HyperCV 项目概述
========================================================================

高光谱数据是一种高维度的数据格式，存储方式包括bil，bsq，bip等。

这个库集成了高光谱图像处理中可能用到的大部分函数。

库中包括两个文件夹，一个core，一个process。

core中是存放核心的声明以及框架函数。

process中是数据处理模块。
